# Roteiro de Implantacao

## Etapa 1. Fechamento

- definir plano
- definir valor de implantacao
- definir mensalidade
- alinhar prazo
- coletar dados do restaurante

## Etapa 2. Coleta de material

- logo
- nome comercial
- telefone e WhatsApp
- endereco
- horario
- cardapio
- formas de pagamento
- taxa de entrega
- pedido minimo
- dominio desejado

## Etapa 3. Configuracao interna

- preencher [docs/CLIENTE_TEMPLATE.md](/C:/Dev/minas/docs/CLIENTE_TEMPLATE.md:1)
- cadastrar tenant em [lib/tenancy.ts](/C:/Dev/minas/lib/tenancy.ts:1)
- configurar variaveis na Vercel
- apontar dominio

## Etapa 4. Personalizacao

- configurar identidade visual
- cadastrar cardapio
- revisar textos
- revisar contatos

## Etapa 5. Validacao

- abrir o dominio final
- testar painel admin
- testar sincronizacao
- realizar pedido de teste
- validar notificacao

## Etapa 6. Entrega

- enviar link final
- passar acesso do admin
- gravar ou fazer treinamento rapido
- orientar uso basico

## Etapa 7. Pos-entrega

- acompanhar primeiros dias
- corrigir pequenos ajustes
- sugerir melhorias futuras
